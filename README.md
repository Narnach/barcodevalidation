# Barcodevalidation

A simple library to validate barcodes

## Installation

Add this line to your application's Gemfile:

    gem 'barcodevalidation'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install barcodevalidation

## Usage

Barcodevalidator.valid?(937179004167)
